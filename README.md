# Travel Memory ✈️

가족·친구와 함께 여행 추억을 모으는 PWA 앱.

## 주요 기능

- 📅 다녀온 여행 타임라인
- 🗺️ 나라/지역별 모아보기
- ✈️ 갈 곳 위시리스트
- 👥 3-4명 그룹 공유 (Google 로그인)
- 📷 iCloud 공유 앨범 연동 (용량 부담 0)
- 📝 일자별 메모 (누가 썼는지 표시)
- ⭐ 별점/한줄평
- 📱 PWA - 홈 화면에 추가하면 앱처럼 사용

## 배포

GitHub Pages에서 자동 배포. URL: `https://[username].github.io/travel-memory/`

## Firebase 보안 규칙

`firebase-rules.json` 내용을 Firebase Console → Realtime Database → 규칙 탭에 붙여넣고 게시.
